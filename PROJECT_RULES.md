# Project Rules - SOC Homelab

## Purpose
This document defines rules for working on the SOC Homelab project.

## AI Assistant Rules

AI assistants may:
- Read documentation
- Analyze configurations
- Suggest improvements
- Explain commands

AI assistants may not:
- Store passwords
- Store API keys
- Commit secrets
- Change critical infrastructure without approval

## Change Management

Before changes:
1. Explain the change
2. Create backup/snapshot
3. Execute change
4. Document result

## Security Principle

Treat this homelab as a professional security environment.
Documentation first, automation second.
